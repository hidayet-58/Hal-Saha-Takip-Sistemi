  Halısaha Takip Sistemi, müşterilerin saha müsaitliklerini görüntüleyip rezervasyon oluşturması ve yöneticilerin saha, müşteri ve rezervasyon süreçlerini yönetmesi amacıyla geliştirildi. Proje ASP.NET Core Web API (.NET) ile oluşturuldu, Entity Framework Core kullanılarak Code-First + Migration yapısı kuruldu ve veriler MSSQL üzerinde tutuldu. Modellerde gerekli DataAnnotation doğrulamaları uygulandı. Migration işlemleri oluşturuldu, veritabanı güncellendi ve proje çalıştırıldığında Swagger üzerinden tüm endpoint’lerin test edilmesi sağlandı. Frontend tarafında Bootstrap kullanıldı ve API tüketimi HttpClient ile gerçekleştirildi. Sistem, halısaha rezervasyon işlemlerinin düzenli ve sorunsuz şekilde yönetilmesi için hazır hale getirilmiştir.
